# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 1.x     | Yes       |

## Reporting a Vulnerability

If you discover a security issue, please email [mail@matthiaswyler.com](mailto:mail@matthiaswyler.com) instead of opening a public issue.

Include:

- A description of the vulnerability
- Steps to reproduce
- Potential impact

You can expect an initial response within 72 hours.
